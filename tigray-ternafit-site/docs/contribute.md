# Contribution Guidelines

## How to Contribute

We welcome contributions from scholars, researchers, community members, and anyone with documented knowledge about Tigray. This knowledge base follows editorial standards to ensure accuracy, neutrality, and credibility.

---

## Before You Contribute

### 1. Read Our Editorial Standards

Review the [Editorial Policy](editorial-policy.md) to understand requirements for accuracy, neutrality, and sourcing.

### 2. Check Existing Content

Search the knowledge base to ensure you're not duplicating existing articles.

### 3. Gather Sources

Prepare citations for all factual claims. Unsourced claims will not be accepted.

---

## Types of Contributions Welcome

### New Content

We particularly need documented content in these areas:

- Historical events with academic sources
- Cultural practices with appropriate citations
- Biographical information for notable figures (with verification)
- Geographic and environmental documentation

### Content Expansion

Many sections are currently under development. You can help by:

- Adding documented information with proper citations
- Providing additional academic references
- Adding context from credible sources

### Source Verification

- Adding credible sources to existing claims
- Identifying stronger academic references
- Fact-checking existing content against primary sources

### Error Correction

- Reporting factual errors with supporting documentation
- Identifying outdated information
- Noting broken source links

---

## Submission Process

### Option 1: Email Submission

1. Write your contribution in plain text or document format
2. Include all sources with full citations
3. Send to: info@ternafit.org
4. Include "Knowledge Base Contribution" in subject line

### Option 2: GitHub (if repository is public)

1. Fork the repository
2. Create a new branch for your contribution
3. Edit or create markdown files
4. Submit a pull request with clear description
5. Respond to editorial feedback

---

## Content Requirements

### Structure

Every contribution should include:

- Clear, descriptive title
- Well-organized sections
- Source citations for all factual claims
- Neutral, academic language

### Citation Format

Use consistent citation format:

**Books**: Author, A. A. (Year). *Title of work*. Publisher.

**Articles**: Author, A. A. (Year). Title of article. *Journal Name*, Volume(Issue), pages.

**Reports**: Organization. (Year). *Report title*. Retrieved from URL

**Web Sources**: Author/Organization. (Date). Title. Website. URL

---

## Quality Standards

### Minimum Requirements

- At least 2-3 credible sources cited
- Clear, neutral language
- Factual accuracy
- Proper formatting

### What We Cannot Accept

- Unsourced claims presented as facts
- Promotional or marketing content
- Politically biased language
- Personal opinions without attribution
- Content that cannot be verified

---

## Editorial Review Process

### 1. Initial Review (1-5 days)

- Basic requirements check
- Preliminary source verification
- Feedback on major issues

### 2. Editorial Review (1-2 weeks)

- Content reviewed for accuracy and neutrality
- Sources verified
- Revisions requested if needed

### 3. Publication

- Final formatting
- Integration into knowledge base
- Contributor credited if desired

---

## Recognition

Contributors may be credited for:

- Major content contributions
- Significant expansions
- Important corrections

Anonymous contributions are also accepted.
