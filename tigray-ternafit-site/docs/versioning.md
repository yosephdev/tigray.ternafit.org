# Versioning and Updates

## Content Versioning

This knowledge base tracks content updates to maintain transparency about when information was added or changed.

---

## Version Information

Major content sections include status indicators:

| Indicator | Meaning |
|-----------|---------|
| **Status: Complete** | Section has undergone full editorial review |
| **Status: In Development** | Section is being actively developed |
| **Status: Under Review** | Section is being reviewed for updates |
| **Last Updated** | Date of most recent substantive change |

---

## Update Practices

### Regular Reviews

- Current events content is reviewed as developments occur
- Statistical data is reviewed periodically for currency
- Source links are checked for accessibility
- Historical content is reviewed less frequently unless new scholarship emerges

### Types of Updates

| Update Type | Description |
|-------------|-------------|
| **Substantive** | New information, revised analysis, significant corrections |
| **Editorial** | Language improvements, formatting, minor clarifications |
| **Source** | New citations added, broken links fixed |
| **Structural** | Page organization, navigation changes |

---

## Change Documentation

Significant revisions are documented, including:

- Addition of new source material
- Correction of factual errors
- Revision of estimates based on new information
- Structural changes to content organization

Minor editorial changes (typos, formatting) are typically not documented.

---

## Transparency Commitment

- Corrections are made openly, not silently
- Significant changes are documented
- Contested information is presented with appropriate context
- Limitations are explicitly acknowledged
- Original versions may be preserved in version history for significant changes
