## About This Project

This knowledge base compiles documented information about the Tigray region of Ethiopia. It is developed by Ternafit, a registered non-profit organization based in Sweden.

**Purpose**: To provide an educational resource for researchers, journalists, students, policymakers, and community members seeking documented information about Tigray.

**Approach**: Content is compiled from international organization reports, academic research, and verified documentation. Sources are cited where available. This is an ongoing project; some sections remain in development.
